# intTaxi 클론코딩  


## 🖥️ 프로젝트 소개
저의 개인프로젝트로 서울시에서 제공하는 택시 서비스 웹페이지를 참고하여 클론코딩 하였습니다. 
기본적인 기능은 그대로 살리되 리뷰, 기업 회원가입 및 로그인, 만족도 평가의 기능 등을 추가하여 웹페이지의 활용도 높이기를 초점에 맞추어 리뉴얼 하였습니다.
